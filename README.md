# Maestro Idle Support Site

Static support + privacy website for App Store URLs.

## Files
- `index.html` - support page
- `privacy.html` - privacy policy
- `styles.css` - shared styles

## Publish with GitHub Pages
1. Create a new public GitHub repository named `maestro-idle-support`.
2. Upload these files to the root of that repo.
3. In GitHub repo settings:
   - Open `Pages`
   - Source: `Deploy from a branch`
   - Branch: `main` and folder `/ (root)`
4. Wait 1-3 minutes for deployment.
5. Your URLs should be:
   - `https://maestrobrandt.github.io/maestro-idle-support/`
   - `https://maestrobrandt.github.io/maestro-idle-support/privacy.html`
